# Black-Scholes Model & Monte Carlo Simulation

This project implements option pricing using the Black-Scholes model and Monte Carlo simulation under the risk-neutral framework. It compares analytical pricing with simulated estimates and explores how stochastic processes translate into numerical methods.

## Overview

- Implementation of the Black-Scholes formula for European call options  
- Monte Carlo simulation of stock price paths using Geometric Brownian Motion  
- Comparison between analytical and simulated option prices  
- Visualization of price paths and profit/loss outcomes  
- Construction and interpolation of a synthetic implied volatility surface  

## Key Concepts

- Risk-neutral pricing  
- Stochastic calculus (Geometric Brownian Motion)  
- Numerical methods (Monte Carlo simulation)  
- Implied volatility and volatility surfaces  

## Tools Used

- Python  
- NumPy, SciPy  
- Matplotlib  
- Pandas  

## Purpose

This project was built to bridge the gap between theoretical models in quantitative finance and their practical implementation. It highlights how concepts such as risk-neutral valuation and stochastic processes are applied to real-world option pricing.
